# Hand Cricket Game

A fun and interactive **Hand Cricket Game** built using **HTML, CSS, and JavaScript**.  
Play against the computer, test your luck, and enjoy the classic childhood game right in your browser!

---

## Features

- Simple and intuitive UI  
- Player vs Computer gameplay  
- Random number generation for fair play  
- Real-time score updates  
- Restart and play again anytime  

---

## Tech Stack

- **HTML** – Structure of the game  
- **CSS** – Styling and layout  
- **JavaScript** – Game logic and interactivity  

---

## How to Play

1. Choose a number between **1 to 6** (your shot/run).  
2. The computer also picks a random number.  
3. If both numbers match → **you’re out!**  
4. Otherwise, your chosen number adds to your score.  
5. Try to score as high as possible before getting out!
